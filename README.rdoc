= Backstab

Backstab is a multiplayer online experimental game. It has an Erlang game engine and Rails frontend serving assets and authorizing users.

== Installation

1. Install Riak:
    <tt>brew install riak</tt>
Use Riak LevelDB backend (see: http://docs.basho.com/riak/1.2.0/tutorials/choosing-a-backend/LevelDB/)

2. Clone repository:
    <tt>git clone https://github.com/oruen/backstab.git</tt>

3. Get submodles:
    <tt>git submodule update --init</tt>

3. Setup Rails app:
    <tt>cd front && bundle install</tt>

4. Then go back and setup Erlang app:
    <tt>cd backend && make all</tt>

== Running

1. Run Riak:
    <tt>riak start</tt>

2. Start Erlang server:
    <tt>cd backend && ./dev/start</tt>

3. Start Rails app:
    <tt>cd front && rails s</tt>
